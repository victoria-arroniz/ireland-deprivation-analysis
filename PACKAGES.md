# R Package Dependencies

Install the required packages from CRAN:

```r
install.packages(c(
  "tidyverse",
  "cluster",
  "MASS",
  "corrplot",
  "dendextend",
  "e1071",
  "pls"
))
```

The report is rendered with [Quarto](https://quarto.org). Install Quarto separately.

**Data source:** [Pobal HP Deprivation Index 2022](https://www.pobal.ie/research-analysis/open-data/) — publicly available, based on the 2022 Irish Census of Population.
